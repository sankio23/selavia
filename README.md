# Selavia — Compendio di Campagna

Sito statico con tutto il materiale della campagna D&D "Selavia" (5e 2024).
Apri **index.html** per la home.

## Pagine
- index.html — home
- bibbia.html — World Bible (canone)
- avvio.html — avvio & sessione zero
- faro.html — animazione "La Notte del Faro Spento"
- guida-pg.html — creazione personaggi
- reliquiario.html — oggetti magici
- archi.html — archi narrativi (master)
- dossier-png.html — schede PNG (master)
- mappa.jpg — mappa delle fasce

## Pubblicare su GitHub Pages
1. Crea un repository nuovo su GitHub (es. `selavia`).
2. Carica tutti i file di questa cartella nella radice del repo.
3. Settings → Pages → Source: branch `main`, cartella `/root` → Save.
4. Dopo ~1 minuto il sito è online su `https://<tuo-utente>.github.io/selavia/`.
